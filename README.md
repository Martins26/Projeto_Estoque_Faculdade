Sistema de Controle de Estoque
Nome: Mario Eduardo dos S. Martins

Descrição do Projeto

Este projeto consiste em um sistema de controle de estoque desenvolvido em Python utilizando a biblioteca Tkinter para criação da interface gráfica.

O sistema permite cadastrar produtos, consultar o estoque, registrar entrada e saída de mercadorias e alertar automaticamente quando um produto estiver com estoque baixo.



Objetivo do Sistema

O objetivo do sistema é auxiliar no gerenciamento de produtos de um estoque de forma simples e intuitiva, permitindo maior controle sobre a quantidade disponível de cada item.


Funcionalidades Implementadas

- Cadastro de novos produtos
- Consulta completa do estoque
- Registro de entrada de produtos
- Registro de saída de produtos
- Alertas automáticos de estoque baixo
- Interface gráfica utilizando Tkinter
- Pop-ups de confirmação e erro com MessageBox
- Atualização automática do estoque e alertas


Instruções de Execução do Código
